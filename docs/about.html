About Our Team

How We Work

The "Stay in GitHub" Rule: All project work lives inside GitHub. Documentation, planning, and task tracking all happen in the repository and GitHub Projects board.

We follow strict documentation guidelines — all docs are written in Markdown, kept short and scannable, and each file focuses on one main topic. Weekly reports are filed from Week 01 onwards in the Weekly/ folder.
